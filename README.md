# Precision Medicine Challenge

## Dataset
You are provided with a CSV containing simulated patient data with which to address the prompt below.

## Data Dictionary

| Variable Name | Description |
|---------------|-------------|
| patient_id | Unique identifier for each patient |
| site_id | Unique identifier for each hospital |
| D1-D4 | Diagnoses at the time they were evaluated for treatment (1 indicates patient diagnosed with condition) |
| Y1-Y5 | Clinical Outcomes (1 indicates a bad outcome) |
| W | Treatment (1 indicates patient received treatment) |
| X1-X80 | Patient characteristics at the time when they were evaluated for treatment |

## Prompt
Which patients with condition D1 would you recommend be given treatment W to reduce their risk of outcome Y1? 

## Submission
Push your solution to a private GitHub repo.

Add `gregorywatson` to the repository.
